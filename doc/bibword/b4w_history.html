Bibtex4Word - Revision History

V2835 2013/03/23

  • Changed search character from : to @ to avoid conflicts with BibTeX key names

V1408 2012/02/10

  • Fixed bug in \pm code and changed to SVN (hence the large version number)

V1.92 2011/07/21

  • Fixed compile error under 64-bit Office version

V1.90 2011/07/09

  • Added BIBTEMP environment variable

V1.89 2011/07/09

  • Added BIBEXE environment variable and improved error diagnostics

V1.87 2011/07/07

  • Fixed error in the reported number of citations in the log file

V1.86 2011/07/0710

  • Fixed error to make the /b style option work with .docx files.

V1.85 2011/07/03

  • Modified the code to work on 64-bit systems

V1.84 2010/06/05

  • Added the /m style flag to control the number of cross references needed to trigger inclusion of the parent entry

V1.83 2009/08/10

  • Fixed punctuation bug for styles that use the mciteplus package.

V1.82 2009/08/09

  • Added the # style flag and partial support for styles that use the mciteplus package.

V1.81 2009/08/06

  • Fixed bug that prevented subscript mode from being reset when exiting math mode.

V1.80 2009/07/19

  • Added interpreter for "bibAnnoteFile" command used in "plos" style..

V1.79 2009/07/17

  • Fixed bug (for the second time) which meant that the font colour in a citation was not preserved when updating.

V1.78 2009/07/17

  • Added & style flag to select the form of "and" in the author list for harvard styles like agsm.
  • Fixed bug which meant that the font colour in a citation was not preserved when updating.

V1.77 2009/07/12

  • Added support for elsarticle-harv style.

V1.76 2009/04/24

  • Added support for \textsc latex command.

V1.75 2009/04/20

  • Added the p option to define the code page of the BibTeX database file. Using the p option alone (without giveing a code page) will use UTF8 encoding and allow the complete range of unicode characters.
  • Now correctly decodes UNIX and Mac line ending formats in the BBL file.

V1.74 2009/04/14

  • Added the 2z option to control whether the name of the BibTeX database file is stored as a relative path with respect to the document.

V1.73 2009/04/13

  • Fixed some errors in the handling of accents.

V1.72 2009/04/12

  • Made the display of citations neater when they are originally entered into the text.

V1.71 2009/04/12

  • Added option to replace URLs with <url> but still have them as hyperlinks.

V1.70 2009/04/11

  • Insert discretionary hyphens after each "/" in a URL to allow them to break across lines. Style flag 3y will suppress this.

V1.68 2009/04/10

  • Removed all non-ascii characters from the file so that the macro should now work correctly for non-Western locales.
  • Fixed Latex mapping of single quotes.

V1.67 2009/04/10

  • Fixed error that prevented one bibliography entry from citing another (as happens when the crossref field is used).

V1.66 2009/04/01

  • Supported \relax command
  • Changed style commands to //t and //b to avoid ambiguity.
  • Improved the behaviour of the /t style table option.

V1.65 2009/03/29

  • Added support for Unicode characters using the \unichar{...} sequence.

V1.64 2009/03/29

  • Fixed bug when a special character ( like _ ) is used as a delimiter for \verb
  • Fixed bug when a special character is used in a label
  • Fixed bug preventing the old bibliography from being deleted when it is updated if the labels are hidden using the l flag.

V1.63 2009/03/28

  • Added support for mathematical expressions and Greek letters

V1.62 2009/03/28

  • Turned off screen updating while bibliography is rewritten to speed things up.
  • Cleared the Undo list when updating the bibliography in order to prevent a warning message from Word about having insufficient memory.

V1.61 2009/03/28

  • Added support for \verb (verbatim mode). This gives the only real way of entering a ~ character.

V1.60 2009/03/27

  • Added mathematical symbols and extended the range of accents

V1.59 2009/03/25

  • Fixed the Visual Basic code for handling accents so that it does not include characters with ASCII codes >127 which causes havoc with chinese installations. I hope this is now fixed but cannot easily tell.

V1.58 2009/03/25

  • Prevented ~ from being converted to a non-breaking space in URL and DOI fields.

V1.57 2009/03/16

  • Fixed error in agsm style when label contains \em (emphasised text)
  • Moved \harvardand interpretation to a more rational place in the code

V1.56 2009/03/14

  • Added support for Harvard styles like agsm
  • Improved stability of the formatting of citations when updating the bibliography
  • Fixed a bug that meant the Latex commands were case sensitive
  • Added support for various year formatting styles

V1.55 2008/04/21

  • Added the + flag to allow the use of bibtex8
  • Added the /t flag to generate a style table

V1.54 2007/09/08

  • Hide the command prompt window when calling Bibtex

V1.53 2007/09/08

  • Improved the blocking call to Bibtex to prevent it hanging on some systems

V1.52 2007/09/08

  • Fixed the fix of V1.51

V1.51 2007/09/08

  • Fixed bug causing errors in continuation lines within a long URL

V1.50 2007/09/08

  • Added h flag for clickable hyperlinks

V1.40 2007/07/14

V1.30 2007/06/30

  • Added support for natbib styles including URL and DOI hyperlinks

V1.20 2007/06/24

  • Fixed bug relating to spaces in bibtex file

V1.12 2006/03/09

  • Fixed errors in Add function

V1.10 2006/03/03

  • Implemented \quotation environment (for annotation style)
  • Implemented \penalty command (for is-plain style)

V1.1 2006/02/12
